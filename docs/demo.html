Data Engineering Showcase

Secure Login Events Ingestion

A privacy-aware ingestion system that receives login telemetry, validates contracts, tokenizes PII, quarantines bad events, loads an analytics-ready model, and records batch-level audit evidence. The demo is intentionally framed as a production data platform, not a toy script.

Reference Data Platform Flow

IngestionSQS

Queue-backed ingestion with bounded batch controls, visibility timeout, and local cloud emulation.

PrivacyHMAC

PII fields are secret-salted and irreversibly tokenized before they reach curated tables.

ReliabilitySLA

Idempotent replay, quarantine, reject-rate tracking, freshness checks, and batch audit evidence.
